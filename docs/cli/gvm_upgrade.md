## gvm upgrade

将 GVM 升级到最新版本

### 使用方法

```bash
gvm upgrade [flags]
```

### 说明

此命令会检查最新 release，并自动下载适合当前平台的升级包，替换当前安装的 `~/.gvm/gvm` 二进制文件。

### 使用示例

```bash
# 升级到最新版本
gvm upgrade

# 查看当前版本
gvm --version
```

### 升级流程

1. 检查 GitHub Releases 获取最新版本
2. 下载对应平台的压缩包
3. 替换 `~/.gvm/gvm` 二进制文件
4. 清理临时文件

### 交互界面

升级界面会展示以下信息：

- 当前版本与目标版本
- 当前平台（例如 `darwin/arm64`）
- 匹配到的升级包文件名
- 下载进度条和已下载大小
- 当前步骤、完成步骤和失败步骤

升级成功后，界面会提示执行 `gvm --version` 验证新版本。

### 注意事项

- 升级过程中不影响已安装的 Go 版本
- 升级命令始终升级到最新版本，不支持选择指定版本
- 建议定期升级以获取最新功能和修复
